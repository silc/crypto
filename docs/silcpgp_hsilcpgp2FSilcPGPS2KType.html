SilcPGPS2KType

NAME

    typedef enum { ... } SilcPGPS2KType;

DESCRIPTION

String-to-key (S2K) specifier types. These define how the passphrase is converted into encryption and decryption key.

SOURCE

typedef enum {
  SILC_PGP_S2K_SIMPLE             = 0,   /* Simple S2K */
  SILC_PGP_S2K_SALTED             = 1,   /* Salted S2K */
  SILC_PGP_S2K_ITERATED_SALTED    = 3,   /* Iterated and salted S2K */
} SilcPGPS2KType;
