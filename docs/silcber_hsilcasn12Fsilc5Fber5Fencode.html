silc_ber_encode

SYNOPSIS

    SilcBool
    silc_ber_encode(SilcBuffer ber, SilcBerClass ber_class,
                    SilcBerEncoding encoding, SilcUInt32 tag,
                    const unsigned char *data, SilcUInt32 data_len,
                    SilcBool indefinite);

DESCRIPTION

Encodes a BER data block into the `ber', which must already have sufficient space allocated. Caller can use silc_ber_encoded_len function to determine how much to allocate space before calling this function. If the `indefinite' is TRUE then the BER block will not include the length of the data in the BER block.
