Key Manager and Repository

DESCRIPTION

SILC Key manager and repository is a generic public key and certificate manager which allows fast and versatile ways to store, retrieve and validate public keys and certificates.

SILC Key Repository is thread safe. Same key repository context can be safely used in multi threaded environment.
