RNG Interface

DESCRIPTION

SILC Random Number Generator is cryptographically strong pseudo random number generator. It is used to generate all the random numbers needed in the SILC sessions. All key material and other sources needing random numbers use this generator.

The interface provides functions for retrieving different size of random number and arbitrary length of random data buffers. The interface also defines Global RNG API which makes it possible to call any RNG API function without specific RNG context.
