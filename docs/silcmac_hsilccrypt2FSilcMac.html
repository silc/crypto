SilcMac

NAME

    typedef struct SilcMacStruct *SilcMac;

DESCRIPTION

This context is the actual MAC context and is allocated by silc_mac_alloc and given as argument usually to all silc_mac_* functions. It is freed by the silc_mac_free function.
