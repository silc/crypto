SilcPublicKeyIdentifier

NAME

    typedef struct { ... } *SilcPublicKeyIdentifier,
                            SilcPublicKeyIdentifierStruct;

DESCRIPTION

This structure contains the SILC Public Key identifier. Note that some of the fields may be NULL.

SOURCE

typedef struct SilcPublicKeyIdentifierObject {
  char *username;
  char *host;
  char *realname;
  char *email;
  char *org;
  char *country;
  char *version;
} *SilcPublicKeyIdentifier, SilcPublicKeyIdentifierStruct;
