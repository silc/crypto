MP Integer Interface

DESCRIPTION

SILC MP Library Interface. This interface defines the arbitrary precision arithmetic routines for SILC. The interface is generic but is mainly intended for crypto usage. This interface is used by SILC routines that needs big numbers, such as RSA implementation, Diffie-Hellman implementation etc.
